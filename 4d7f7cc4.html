Posted on 2020-10-15 In 编程技术

Markdown 标题

  1. 使用 = 和 - 标记一级和二级标题
  2. 使用#标记
    • 使用 #表示1-6级标题，一级标题对应一个# ,二级标题对应两个#，以此类推。

Markdown 段落

Markdown 段落没有特殊的格式，直接编写文集就好，段落的换行是使用两个一个空格加上回车。

字体

- Markdown 可以使用 `*` 或者`_`表示斜体
- Markdown 可以使用 `**`或者 `__` 表示粗体
- Markdown 可以使用 `***`或者 `___`表示粗斜体

分割线

  • Markdown可以使用***表示分割线
  • Markdown 可以使用--- 表示分割线

删除线

  • 如果段落上的文字要添加删除线，只需要在文字的两端加上两个波浪线 ~~即可.例如： 删除线

下划线

  • 下划线可以通过HTML的<u>标签来实现， 例如: 下划线

脚注

  • 脚注是对文本的补充说明。Markdown脚注的格式如：[^要注明的文本]

    例如：[^RUNOOB]

    [^RUNOOB]: 菜鸟教程 – 学的不仅是技术，更是梦想

Markdown 列表

Markdown 支持有序列表和无序列表

  • 无序列表使用*或者+ 或者- 作为列表标记，这些标记后面要添加一个空格，然后再填写内容
  • 有序列表使用数字加上.来表示，如1.

列表嵌套

列表嵌套 只需再子列表中的选项前面添加四个空格即可。

Markdown 区块

Markdown区块引用是再段落开头使用>符号，然后紧跟一个空格符号。

区块是可以嵌套的，一个 >符号是最外层，两个>符号是第一层嵌套，以此类推。

区块里面可以嵌套列表

列表里面可以嵌套区块， 需要再 > 前添加四个空格的缩进。

Markdown 代码

  • 如果是再段落上的一个函数或片段的代码，可以使用反引号(`) 把它包起来，例如printf()函数

  • 代码区块 可以使用 三个反引号 包裹一段代码，并指定一种语言（也可以不指定）
